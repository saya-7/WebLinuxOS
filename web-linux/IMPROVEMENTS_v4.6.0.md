## 新增功能

### 1. JSON Schema 验证器 (JSONSchemaValidator)
全新的 JSON Schema 验证工具，提供以下功能：
- **完整的 Schema 验证**: 支持类型检查、属性验证、必填字段等
- **实时验证**: 快速验证 JSON 数据是否符合 Schema
- **预设模版**: 用户数据、产品列表、API 响应等预设
- **友好的错误提示**: 清晰显示验证失败位置和原因
- **性能统计**: 显示验证耗时
- **格式化工具**: 支持 JSON 格式化

### 2. Markdown 到 HTML 转换器 (MarkdownToHTML)
功能强大的 Markdown 转换工具：
- **语法支持**: 标题、列表、链接、粗体、斜体、代码块、引用等
- **实时预览**: 三个标签页（编辑器、预览、HTML 源码）
- **导出选项**: 复制 HTML 或下载完整 HTML 文件
- **预设模版**: README、博客文章、简历等常用文档模版
- **美化输出**: 包含完整的 CSS 样式，生成专业美观的 HTML

## 使用指南

### JSON Schema 验证器
1. 加载预设或输入自定义 JSON Schema
2. 输入待验证的 JSON 数据
3. 点击"验证"按钮开始验证
4. 查看验证结果和错误详情

### Markdown 到 HTML 转换器
1. 在编辑器中编写 Markdown 文本
2. 切换到"预览"查看效果
3. 点击"复制 HTML"或"下载"保存结果

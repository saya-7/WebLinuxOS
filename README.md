# WebLinuxOS

基于 Web 的 Linux 桌面环境模拟器，使用 React + TypeScript + Vite 构建。

## 特性

- 🖥️ 完整的 Linux 桌面环境体验
- 📱 56+ 功能齐全的应用程序
- 🪟 窗口管理系统（拖拽、调整大小、最小化、最大化）
- 📁 模拟文件系统
- 💻 终端仿真器
- 🎨 SVG 图标系统
- 🌓 暗色/亮色主题
- 🖼️ 壁纸切换

## 应用分类

| 分类 | 应用 |
|------|------|
| 系统 | 文件管理器、终端、系统监视器、设置、软件包管理器、软件中心等 |
| 办公 | 文本编辑器、日历、记事本、PDF 查看器、电子表格、演示文稿等 |
| 互联网 | 浏览器、邮件客户端、即时通讯、地图 |
| 多媒体 | 图片查看器、音乐播放器、视频播放器、画图、摄像头等 |
| 工具 | 计算器、时钟、天气、截图工具、密码管理器等 |
| 开发 | 代码编辑器、命令参考 |
| 游戏 | 贪吃蛇、俄罗斯方块 |

## 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite
- **状态管理**: Zustand
- **图标**: 自定义 SVG 组件

## 开发

```bash
cd web-linux
npm install
npm run dev
```

## 构建

```bash
cd web-linux
npm run build
```

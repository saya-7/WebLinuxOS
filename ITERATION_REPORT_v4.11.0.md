## 部署流程

### GitHub Actions 自动部署
当代码推送到 main 分支时，GitHub Actions 工作流会：
1. 检出代码
2. 设置 Node.js 20 环境
3. 使用 npm ci 安装依赖
4. 执行 npm run deploy 进行构建
5. 设置 Pages 部署环境
6. 上传构建产物到 Pages
7. 自动部署到 GitHub Pages

### 手动部署
也可以通过本地构建后推送到仓库来触发部署：

```bash
cd web-linux
npm run deploy
git add .
git commit -m "更新部署"
git push origin main
```
